/* LISTAS */

Tipos de lista:

1. Lista ordenada. <ol> <li> </li> </ol>
2. Lista desordenada. <ul> <li> </li> </ul>
3. Lista por definición. 
<dl> 

<dt> </dt> --> Término.
<dd> </dd> --> Definición.
</dl>

/* LINKS */

Para rediregir a una nueva pestaña utilizamos el atributo target = "_blank"
en nuestra etiqueta <a> </a>.

/*  IMÁGENES */

Rutas absolutas: indicamos carpeta por carpeta donde se encuentra
nuestro fichero en nuestra computadora.

Rutas relativas: indica donde se encuentra nuestro fichero apartir de 
nuestro proyecto.
